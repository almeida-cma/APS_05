Valor total da conta (R$) em um único campo numérico.

Calcular: Dinheiro (10% off), PIX (5% off), Débito (integral), Crédito 1x/2x/3x (2%, 4%, 7% juros).

📤 RF03 - Saída

Exibir 6 cards com valores calculados.

    def button_calcular_click(self, **event_args):

        # 1. ENTRADA (Como o TextBox é type="number", o Anvil já entrega o número ou None)
        valor = self.text_box_valor.text

        # 2. VALIDAÇÕES LIMPAS
        if valor is None:
            self.exibir_erro("⚠️ Digite o valor da conta!")
            return

        if valor <= 0:
            self.exibir_erro("❌ Valor deve ser maior que zero!")
            return

        # 3. CÁLCULOS
        def fmt(v):
            return f"R$ {v:.2f}".replace('.', ',')

        # DINHEIRO (10% desconto)
        self.label_dinheiro.text = fmt(valor * 0.90)

        # PIX (5% desconto)
        self.label_pix.text = fmt(valor * 0.95)

        # DÉBITO (integral)
        self.label_debito.text = fmt(valor)

        # CRÉDITO 1x (2% juros)
        self.label_credito1.text = fmt(valor * 1.02)

        # CRÉDITO 2x (4% juros) - mostra o valor da parcela
        total2 = valor * 1.04
        self.label_credito2.text = fmt(total2 / 2)

        # CRÉDITO 3x (7% juros) - mostra o valor da parcela
        total3 = valor * 1.07
        self.label_credito3.text = fmt(total3 / 3)


    def exibir_erro(self, mensagem):
        labels = [
            self.label_dinheiro, self.label_pix, self.label_debito,
            self.label_credito1, self.label_credito2, self.label_credito3
        ]
        for label in labels:
            label.text = mensagem
