📘 UML · Aprendizado visual e completo

🎯 Unified Modeling Language

Domine os 7 diagramas essenciais com exemplos grandes, notações corretas e 10 desafios práticos.

📌 1. Diagrama de Caso de Uso

Funcionalidades do ponto de vista do usuário. Atores (bonecos palito) e casos de uso (elipses).

🎯 Cliente pode Pesquisar, Comprar (e ao comprar, verifica estoque). Gerente cadastra produtos e gerencia vendas.
🏛️ 2. Diagrama de Classes

Estrutura estática: classes, atributos, métodos e relacionamentos.

🔗 Vários Produtos pertencem a uma única Categoria (relacionamento 1 para muitos).
⏱️ 3. Diagrama de Sequência

Ordem temporal das mensagens entre objetos.

📞 Passo a passo: Usuário → Sistema → Banco → retorno de dados.
⚙️ 4. Diagrama de Atividades

Fluxo de controle, decisões e atividades.

🔀 Fluxo: Início → Fazer pedido → verifica pagamento → aprovado ou cancelado.
🔄 5. Diagrama de Estado

Estados de um objeto e transições entre eles.

📦 Estado de um Pedido: Rascunho → Enviado → Entregue. Cancelamento possível a qualquer momento.
🧩 6. Diagrama de Componentes

Organização dos componentes de software e dependências.

📡 Frontend consome API REST, que acessa o banco de dados SQL.
🖥️ 7. Diagrama de Implantação

Arquitetura física: nós de hardware e software implantado.

🌐 O usuário acessa via navegador → Servidor Web na nuvem → Banco de dados remoto.
